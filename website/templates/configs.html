Instructions


  1. Go to Settings tab and configure the inner-connections between physical parts such as Raspberry-Arduino Serial Communication, PiCamera and Servo Motors.

  2. Test the connections. Make sure everything works correctly and submit the changes you made.

  3. Go to Sort tab, and create new Sorting Session.

  4. Follow the instructions there to complete the sorting process

  5. Enjoy using the Sorting Machine


Important notes:

The Settings are set to Optimized Defaults

Many settings are not able to be changed because of component's dependencies
